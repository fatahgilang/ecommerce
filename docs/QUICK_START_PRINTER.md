# 🚀 Quick Start - Thermal Printer

## 3 Langkah Mudah

### 1️⃣ Install QZ Tray (5 menit)
```
1. Download: https://qz.io/download/
2. Install aplikasi
3. Jalankan QZ Tray (icon muncul di system tray)
```

### 2️⃣ Setup Printer (2 menit)
```
1. Buka aplikasi Toko Makmur
2. Klik icon "Printer" di navigation
3. Klik "Refresh Printer"
4. Pilih printer thermal Anda
5. Klik "Test Print"
```

### 3️⃣ Mulai Cetak! (Instant)
```
1. Customer checkout di frontend
2. Pesanan berhasil
3. Klik "Ya" saat ditanya "Cetak struk?"
4. Struk langsung keluar! 🎉
```

## ✅ Checklist

Sebelum mulai, pastikan:
- [ ] QZ Tray terinstall dan berjalan
- [ ] Printer thermal terhubung (USB/Bluetooth)
- [ ] Driver printer terinstall
- [ ] Kertas thermal tersedia
- [ ] Printer sudah dipilih di aplikasi
- [ ] Test print berhasil

## 🎯 Demo


## 🆘 Troubleshooting Cepat

| Masalah | Solusi |
|---------|--------|
| QZ Tray tidak terdeteksi | Restart QZ Tray, refresh browser |
| Printer tidak muncul | Install driver, restart komputer |
| Struk tidak keluar | Cek kertas, cek koneksi printer |
| Format berantakan | Pastikan printer 58mm, update driver |
